# Customer Churn Prediction Using Deep Learning Techniques

## Overview

Customer Churn Prediction System is a web-based application developed using Django, Machine Learning, and Deep Learning techniques. The system analyzes customer data and predicts whether a customer is likely to leave a service or continue using it. This helps organizations improve customer retention and business performance.

## Objectives

* Predict customer churn accurately.
* Identify customers at risk of leaving.
* Support decision-making using data-driven insights.
* Improve customer retention strategies.

## Features

* Customer churn prediction
* Machine Learning and Deep Learning models
* User-friendly web interface
* Real-time prediction
* Email notification system
* Data preprocessing and analysis
* Secure authentication and login

## Installation

### Clone Repository

```bash
git clone https://github.com/Shankarghuge/churn-prediction.git
```

### Move to Project Directory

```bash
cd churn-prediction
```

### Install Dependencies

```bash
pip install -r requirements.txt
```

### Run Migrations

```bash
python manage.py migrate
```

### Start Development Server

```bash
python manage.py runserver
```

### Open Browser

```text
http://127.0.0.1:8000/
```
